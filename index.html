TweePLayer Event Creator by Tomorrowish

A Chrome extension based on Michromeformats that reads event microformats on a page and allows you to add them as TweePLayer events.

The TweePLayer Event Creator is based on the Michromeformats project. It is being changed to focus only on event formated data online. Tools to add the event to TweePLayer's social media content harvester are also being added.
